# Spruce Budworm Radar Segmentation — Research Progress

Interactive single-page dashboard tracking three experiments that select the architecture, input channels, prediction target, and loss function for segmenting spruce budworm dispersal plumes in XAM weather radar.

## Experiments
- **Experiment 1** — Architecture and loss comparison (10 runs)
- **Experiment 2** — Terrain channel ablation (12 runs)
- **Experiment 3** — Target and loss selection (18 runs)
